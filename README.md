# Flutter Skills

This repository collects reusable "skills" for Flutter applications. Each skill is packaged in its own directory under the `skills/` folder and includes documentation, examples, and integration notes.

## Skills Directory

- [flutter-adaptibility-skill](skills/flutter-adaptibility-skill/SKILL.md): Skill for adapting Flutter UI and behavior to different device sizes and accessibility needs. See the skill documentation for usage and examples.

## How to use a skill

1. Open the skill's directory under `skills/`.
2. Read the `SKILL.md` file for details, examples, and integration instructions.
3. Copy the implementation or follow the integration notes into your Flutter project.

## Contributing

- Add a new folder under `skills/` with a clear name.
- Include a `SKILL.md` describing the problem, solution, API, and examples.
- Optionally add example code and tests.
